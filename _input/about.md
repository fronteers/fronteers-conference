Fronteers Conference is organised by volunteers of Fronteers, the Dutch professional association for front-end developers.

{% set src = '/img/crowd.jpg' %}
{% include '_partials/hero.njk' %}

Besides organising Fronteers Conference, the professional association also runs regular meet-ups and workshops and an online job board. Fronteers has roughly 220 members, mainly from The Netherlands. See also: [About Fronteers](https://fronteers.nl/about)
